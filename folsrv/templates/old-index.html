upgrade-ubuntu.sh
Use this to upgrade form 10.04 to 12.04.
N.B. This script performs a backup, so you should not do one first as it will complain. You can force it to overwrite it with the -f option.

Steps to upgrade the script. Note the following carefully:

  1. Run the script, carefully noting any instructions given
  2. When prompted to reboot. DO NOT REBOOT
  3. Press x, then type reboot on the command line.
  4. Rerun the script a second time to finish the process

Tech Support Tools
This is a tarball that uncompresses to a directory containing a bunch of usefull shell scripts for tech support purposes. The main ones of interest are ts_network_backup which automates the network backup process and ts_identify_backups which is used in conjunction with it (given a ticket number it will identify the directory it was backed up to.
The latest version can always be found at:
http://tsbackup/tstools.tar.gz.
The current version is tstools.0.1.13.tar.gz.
Eventually it will be come a deb that be installed via apt-get form the freegeek archives. N.B. This is still beta quality, bugs may be present though it is definitely usable. Bug reports are welcome via email to bugs@todo.freegeek.org (or the bugs queue in RT). Please include any output/error messages and the version you were using, thank you.
