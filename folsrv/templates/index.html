Tech Support Tools
This is a package containing a bunch of usefull shell scripts for tech support purposes. The main ones of interest are ts_network_backup which automates the network backup process and ts_identify_backups which is used in conjunction with it (given a ticket number it will identify the directory it was backed up to.
The latest stable version is packaged as a deb in the Free Geek archive so you should be able to install it using apt-get apt-get install fg_tstools
The current version is {{current_version}}
Bug reports are welcome via email to bugs@todo.freegeek.org (or the bugs queue in RT). Please include any output/error messages and the version you were using, thank you.
